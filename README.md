# audiolens-models-comparison

Note: Model weights and result files are not included in this repository due to their size. When you run the scripts for the first time, they will automatically download the required model weights from HuggingFace Hub and save them to the local models/ directory. Evaluation results are generated during execution and saved to the results/ folder. Both directories are excluded from version control via .gitignore. To reproduce the full evaluation from scratch, simply run the scripts in order - everything will be fetched and recorded automatically.
